Emoji preview SVG assets
========================

These small transparent Face with Tears of Joy assets are used only by the emoji
font picker preview, so opening the picker does not require rendering the
preview glyph through a full emoji font.

Included source artwork:

- `noto.svg`: googlefonts/noto-emoji `emoji_u1f602.svg`
- `noto-mono.svg`: same Noto source artwork; the upstream monochrome font does
  not publish a separate SVG asset
- `blobmoji.svg`: C1710/blobmoji `emoji_u1f602.svg`
- `fluent.png`: microsoft/fluentui-emoji 3D PNG
- `fluent-flat.svg`: microsoft/fluentui-emoji Flat SVG
- `fluent-mono.svg`: microsoft/fluentui-emoji High Contrast SVG
- `twemoji.svg`: jdecked/twemoji `1f602.svg`
- `openmoji.svg`: hfg-gmuend/openmoji `1F602.svg`
- `emojitwo.svg`: EmojiTwo `1f602.svg`
- `tossface.svg`: toss/tossface `u1F602.svg`
